# Flipkart-Customer-Satisfaction-Prediction

Machine Learning project for predicting customer satisfaction scores using Flipkart customer support data.

# Flipkart Customer Satisfaction Prediction

## Project Overview
This project predicts customer satisfaction (CSAT) scores using customer support interaction data from Flipkart.

## Dataset Features
- Support Channel
- Category
- Product Category
- Agent Details
- Handling Time
- Customer Information

## Machine Learning Models
- Decision Tree
- Random Forest

## Results
Random Forest achieved the highest accuracy and provided valuable insights into customer satisfaction drivers.

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
