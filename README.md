# Flipkart Customer Satisfaction Prediction Using Machine Learning

## Project Overview

Customer satisfaction is a critical factor in the success of e-commerce platforms. This project focuses on predicting Customer Satisfaction (CSAT) scores using customer support interaction data from Flipkart.

The project includes data cleaning, exploratory data analysis, feature engineering, and machine learning model development using Random Forest Classification.

## Objectives

* Analyze customer support interaction data.
* Identify factors affecting customer satisfaction.
* Build a machine learning model to predict CSAT scores.
* Generate business insights for improving customer support services.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Machine Learning Model

* Random Forest Classifier

## Results

The Random Forest model was trained to predict customer satisfaction scores and evaluated using accuracy score, classification report, and confusion matrix.
